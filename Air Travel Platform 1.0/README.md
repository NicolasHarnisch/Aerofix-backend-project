# Airline Ticket Reservation System / Version 1.0 🎟️

## 📘 README (Português)

Este módulo mostra a versão inicial da plataforma AEROFIX, com funcionamento em console:

- Visualização de mapa de assentos (livres vs ocupados)
- Escolha de classe: Econômica ou Executiva
- Reserva de assentos durante a sessão
- Suporte a grupos/famílias (2 a 5 pessoas)
- Restrição: em classe econômica, assentos A e F não podem ser reservados
- Sugestão automática de assentos adjacentes

### Como compilar e executar

1. Abra terminal em `Air Travel Platform 1.0/`
2. Compile:
   - `g++ -std=c++17 -o app Main.cpp`
3. Execute:
   - `./app` ou `app.exe`

## 📘 README (English)

This module represents AEROFIX version 1.0 with a console-based reservation system:

- Seat map visualization (available vs occupied)
- Class selection: Economy or Business
- Reserve seats during runtime session
- Family bookings (2 to 5 people)
- Economy restriction: seats A and F blocked
- Automatic adjacent seat recommendation

### Build and run

1. Open terminal at `Air Travel Platform 1.0/`
2. Compile:
   - `g++ -std=c++17 -o app Main.cpp`
3. Run:
   - `./app` or `app.exe`
